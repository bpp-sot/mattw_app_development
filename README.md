
# App Development Project

### 1. Project Overview

- This application helps users compare the contents of two Excel spreadsheets and display the differences between them.
- It is aimed at employees at the organisation who routinely have to compare Excel files manually as part of a comparison process.
- It will help to speed up the current manual process of comparing excel files, whilst also improving the accuracy and consistency of the comparisons


---

### 2. Setup & Deployment Instructions

Explain:

- To clone the project, navigate to the Github homepage associated with this project: https://github.com/bpp-sot/mattw_app_development
- Copy the HTTPS link from the green "Code" button in Github

- To run the app locally, run the following command in Github Codespaces Terminal:

```bash
npm run dev
```

- Click on the local host link that is returned, this will open a new window, where the app is running
